1. 错误码(errorCode)

xxyyzz

  • 类型：Number
  • xx 业务模块类型，如会话管理模块（10）
  • yy 子业务模块类型，如登陆子模块（01）
  • zz 错误类型，如用户名或密码错误（01）

通用错误码：

  • 0 ：处理成功（OK）
  • 500 ：服务器内部错误（server error）

2. 会话 - 10yyzz

2.1. 登陆 - 1001zz

2.1.1. 请求

方法 URL 参数
POST /api/session/login clientName、clientPassword

示例：

// POST /api/session/login
{
  "clientName": "admin",
  "clientPassword": "7410"
}

2.1.2. 响应

成功：

{
  "errorCode": 0,
  "reason": "OK",
  "result": {
    "clientId": "1",
    "clientName": "admin",
    "clientPassword": "7410", // 明文存储密码
    "createtime": "2018-09-17T02:14:18.523Z",
    "modifiedtime": "2018-09-17T02:14:18.523Z",
    "clientTrueName": "超级管理员",
    "clientUserSex": -1,
    "allowLogin": 1,
    "clientUserIdCard": "",
    "clientUserPhoneNum": "",
    "clientUserEmail": "",
    "createuserid": "",
    "createusername": "",
    "modifieduserid": "",
    "modifiedusername": "",
    "isdeleted": 0
  }
}

失败：

{
  "errorCode": 100101,
  "reason": "clientName or clientPassword is invalid."
}

2.2. 登出 - 1002zz

2.2.1. 请求

方法 URL 参数

成功：

  "reason": "OK"

3. 用户管理 - 11yyzz

3.1. 用户添加 - 1101zz

方法 URL 参数

示例：

{
  "clientName" : "admin2",
  "clientPassword" : "123",
  "clientTrueName" : "吴钦飞",
  "clientUserEmail" : "w@q.c",
  "clientUserPhoneNum" : "18707127626",
  "clientUserSex" : 0
}

3.1.2. 响应

成功：

{
  "errorCode": 0,
  "reason": "OK"
}

失败：

{
  "errorCode": 110101,
  "reason": "add user fail."
}

3.2. 用户列表 - 1102zz

3.2.1. 请求

方法 URL 参数
POST /api/user/list clientName（模糊查询）、pager（分页相关）

示例：

// POST /api/user/list
{
  "clientName": "a",
  "pager": {
      "pageSize": 10,  // 一页的大小
      "currentPage": 1 // 请求的页数
  }
}

3.2.2. 响应

成功：

{
  "errorCode": 0,
  "reason": "OK",
  "result": {
    "items": [
      {
        "clientTrueName": "超级管理员",
        "clientUserSex": -1,
        "allowLogin": 1,
        "clientUserIdCard": "",
        "clientUserPhoneNum": "",
        "clientUserEmail": "",
        "createuserid": "",
        "createusername": "",
        "modifieduserid": "",
        "modifiedusername": "",
        "isdeleted": 0,
        "clientName": "admin",
        "clientPassword": "7410",
        "clientId": "f56ad4a0-c061-11e8-bc13-574c511256c4",
        "createtime": "2018-09-25T01:25:59.664Z",
        "modifiedtime": "2018-09-25T01:25:59.664Z",
      }
    ],
    "pager": {
      "pageSize": 1,
      "total": 10,
      "currentPage": 1
    }
  }
}

失败：

{
  "errorCode": 110201,
  "reason": "add user fail."
}
